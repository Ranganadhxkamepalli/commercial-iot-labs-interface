Read the Objectives

The Intel® IoT Developer Hub is a web-based service running on the gateway that allows you to have a hands-on sensor-to-cloud experience in a very short time period. It has built-in tutorials to teach you about tools like the visual programming interface called Node-RED* and the Wind River Helix App Cloud, including sample code to develop the app in the cloud. The Intel® IoT Developer Hub provides ways to easily manage the administrative settings of the gateway. The Intel® IoT Developer Hub lets you, as the developer, add packages and develop your apps. When development is complete, you can save a hardened OS image onto a USB flash drive.

By the end of this module, you should be able to:

  • Connect gateway with Helix App Cloud
  • Create graphical view of Sensor data
  • Access repositories and manage packages on gateway
  • Access Gateway Administration tools and resources
  • Getting Familiar with Cloud Commander

Introduction to Intel Iot Dev Hub

  1. Open your browser and go to IP address of your gateway to access IoT Developer Hub.
  2. Login to the Intel® IoT Gateway Developer Hub using root for both the username and the password.
  3. Use the Intel® IoT Gateway Developer Hub to explore and extend the basic capabilities of the Gateway.

Advantages of using Intel IoT Dev Hub

The advantages of using Intel IOT Developer Hub gives you,

  • Live Dashboard
    • Summary view of gateway information
    • Graphical view of sensor data
    • Notification of available OS updates
  • Plug and play sensors
    • Shows live data from connected sensors
    • Easy management of connected sensors via Node-RED
  • Simple access to repositories and management of popular package
    • Simple to add or remove online package repositories
    • Easily list installed packages
    • Quickly add, update, or uninstall packages as needed
  • Gateway administration tools and resources
    • One-touch buttons for gateway actions including: restart, factory reset, upgrade to Pro, update OS, and change password
    • Step-by-step wizard for creating a deployable gateway image
    • Direct access to popular development tools including: Node-RED, LuCI*, Cloud Commander*, and Wind River Helix App Cloud*
  • Clear and simple documentation
    • One location to find how-to tutorials and videos, and quick links to key online content

Helix App Cloud Overview

Helix App Cloud provides an eloquent way to develop applications against the gateway from the cloud. It creates a connection between the gateway and the cloud so you can seamlessly develop, execute and debug code on the gateway from the cloud.

Launch Helix App Cloud

From the administration section of the Intel IoT Developer Hub, click the Launch button under the Helix App Cloud Icon

Copy Registration Code

Copy the Unique ID that is provided into your clipboard (ctrl + c). This Unique ID is used to associate your gateway with the Helix App Cloud service. It will be used after you sign-up, and log back in to verify your email.

Then Click on "Continue to Helix App Cloud"

Register and Verify an HAC Account

Follow the registration steps on Helix App Cloud (if you do not already have an account).

If you already have an account with Windriver or Intel then skip registration step and directly sign in.

It is important that you verify your email within 20 minutes of copying the code, as it will expire. Please verify your email account at this point.

Add New Device

In Helix App Cloud, click "New Device" in the top right corner to add a new device.

Click "Register an existing device using its Registration ID"

Paste in the Unique ID. (previously copied into clipboard). Enter a device name, then click "Register Device".

Helix App Cloud Dashboard

On the Helix App Cloud deshboard, you can check your device is online or offline at the top of the page.

There is cloud based chart available to monitor CPU and Memory of IoT Gateway.

Device Shell will provide an interface to connect your device and provide command line access.
