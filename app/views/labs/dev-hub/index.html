Read the Objectives

The Intel速 IoT Developer Hub is a web-based service running on the gateway that allows you to have a hands-on sensor-to-cloud experience in a very short time period. It has built-in tutorials to teach you about tools like the visual programming interface called Node-RED* and the Wind River Helix App Cloud, including sample code to develop the app in the cloud. The Intel速 IoT Developer Hub provides ways to easily manage the administrative settings of the gateway. The Intel速 IoT Developer Hub lets you, as the developer, add packages and develop your apps. When development is complete, you can save a hardened OS image onto a USB flash drive.

By the end of this module, you should be able to:

  • Connect gateway with Helix App Cloud which is cloud based IDE
  • Create graphical view of Sensor data
  • Access repositories and manage packages on gateway
  • Access Gateway Administration tools and resources
  • Getting Familiar with Cloud Commander
