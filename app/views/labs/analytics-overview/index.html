Video Analytics Overview

Video Analytics section is intended to demonstrate how multiple SDKs can be used together to improve performance for Video Analytic workloads

It provides simple tutorials using OpenCV and explain how the upcoming optimized Intel Computer Vision SDK could work with real code examples

Also includes end-to-end samples of real use cases like Motion Detection and Face Detection to showcase performance. These may be used for initial evaluation, and possibly as an advanced reference.
