Remote Management Overview

Remote Management section is intended to demonstrate the extended power of Intel vPro technology Intel vPro technology is an umbrella marketing term used by Intel for a large collection of computer hardware technologies, including Hyperthreading, Turbo Boost 3.0, VT-x, VT-d, Trusted Execution Technology (TXT), and Intel Active Management Technology (AMT).

Why Remote Management using Intel® vPro?

Future of Retail Store kind of scenarios are starting now. All the sub-systems used in this demands secure remote management capabilities which are fulfilled by Intel® vPro Technology and highly secure Intel Security Products.

Intel® vPro platform is the sum of hardware, BIOS extensions and applications that deliver solutions for a robust and reliable retail applications.

This section gives hands on experience to enable Intel@AMT, browse through Intel® vPro Environment and modes, how to manage remote systems using a desktop tool like Mesh Commander and open source cloud based solution like Mesh Central.
