HDC should be installed on your Ubuntu Gateway system. In case it is not, follow the below steps to install it

Below steps are required only if HDC is not installed. To confirm if HDC files are not there go to /var/lib/iot and check if files agentguid.bin, dxlpolicy.bin and startup.bin are present. If not follow the below install steps.

Download following github repository:

git clone https://github.com/SSG-DRD-IOT/gateway-setup
cd gateway-setup

2. Install the HDC deb package

apt-get install -y $PWD/conf_files/hdc/hdc-2.3.0-x86_64.deb

3. Copy the startup.bin to /var/lib/iot

cp conf_files/hdc/startup.bin /var/lib/iot

4. Run the iot-control program with startup.bin as its parameter.

iot-control -c /var/lib/iot/startup.bin
