Introduction

Intel hardware accelerated codecs are now accessible via FFmpeg* on Linux* systems where Intel® Media Server Studio is installed. The same OS and platform requirements apply, as the new additions to FFmpeg are simple wrappers to bridge the APIs.

Exploration

To get started one has to follow these steps:

  • Install Intel Media Server Studio for Linux (Done)
  • Get the latest FFmpeg source (Done)
  • Configure FFmpeg (Done)
  • Transcode
Observation

Observe the integration of ffmpeg with Intel® Media Server Studio

Learning Outcome

By the end of this module, the participant is expected to understand briefly the working of Intel® Media Server Studio with FFMPEG

Transcode with an accelerated codec such as "-vcodec h264_qsv" on the ffmpeg command line.

Performance boost increases with resolution.

$ffmpeg -i /home/intel/Documents/MSDK/mediasdk.mp4 -vcodec h264_qsv -init_hw_device qsv:hw test.mp4


Demux audio and video from an mp4 container using ffmpeg

$ffmpeg -i test.mp4 -c:v copy -map 0:0 video.mov -c:a copy -map 0:1 audio.aac
