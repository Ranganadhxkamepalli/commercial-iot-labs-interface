Lab Overview

In this section we will learn how to capture video frames from the connected camera using OpenCV and Python

Pre-requisites

  • Make sure you have connected the given Camera to you Intel ® NUC
  • Mount camera at some position where you can see your face when we capture video.
Launch IDLE for Python development
  • Press Windows + S
  • Type IDLE
  • Click on IDLE application
Create new file
  1. Open IDLE
  2. Select File > New Window
  3. A blank file opens up
  4. Type following two lines
  5. import numpy as np
import cv2
  6. Press CTRL + S to save the file
  7. Press F5
Video Capture and Display
  1. Capture the video using cv2.VideoCapture
  2. Video capture method takes one parameter. This parameter is the index number of camera that you want to use. For our workshop, we are using only one camera.. hence It will be 0.
  3. Create a namedWindow to show the output
  4. Show the output on this namedWindow using imshow method
  5. Copy the below code and put it on your file and run your application
  6. Press Esc to exit
  7. cap = cv2.VideoCapture(0)

while(True):
    # Capture frame-by-frame
    ret, frame = cap.read()

    #Set full screen
    cv2.namedWindow("Target", cv2.WND_PROP_FULLSCREEN)
    cv2.setWindowProperty("Target",cv2.WND_PROP_FULLSCREEN,cv2.WINDOW_FULLSCREEN)

    # Display the resulting frame
    cv2.imshow('Target',frame)

    if cv2.waitKey(1) & 0xFF == 27:
        break

# When everything done, release the capture
cap.release()
cv2.destroyAllWindows()
  8. Press CTRL + S to save the file
  9. Press F5
