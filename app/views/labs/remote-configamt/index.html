This guide is intended to be a starting point for testing your Intel ® vPro™ technology systems within your environment.

It will guide you through setting up a "Sandbox" test environment using your own network environment so that you can test your specific use cases on Intel vPro technology that ultimately will help reduce overall support costs and costly service/truck rolls.

The first step in enabling Intel® AMT is to install the Intel® AMT agent. In this lab, we create a USB drive that will install the Intel® AMT agent onto a host system. Then we will verify that the agent is correctly install by going to a web browser and accessing the Intel® AMT firmware based web configuration console.

Start by performing these actions on your Intel IoT Gateway:

  1. Insert the empty USB drive in the USB port on the host system.
  2. Goto C:\Users\<username>\Desktop\Retail\02-RemoteManagement\Configurator_download_package_11.1.0.75\ACU_Wizard folder
  3. Right-click ACUWizard.exe and select Run as administrator.

The Intel® AMT Configuration Utility appears.

Select Create Settings to Configure Multiple Systems.

The following screen appears

Click the Tools drop-down menu and select Prepare a USB Key for Manual Configuration

The following screen appears

Select the type of Intel vPro system that you will be configuring: Mobile Systems or Desktop Systems.

Note: Some Small Form Factor (SFF) desktops use mobile chipsets. For example, QM87 is a mobile Intel vPro chipset noted by the "M," whereas the desktop equivalent chipset is the Q87.

Select All systems are version 7.0 and higher.

In the Configuration Settings section:

  • Old Intel MEBX Password: admin

    Intel SCS always puts the default password of unconfigured systems ("admin") in this field.

    For the present workshop new MEBX password is "Intel@1234"

    If this is not the password currently defined in Intel MEBX, enter the correct password.

    Note: If you have lost your Intel MEBX password, you will need to consult the OEM manual for clearing the CMOS battery to reset the password to "admin".
  • New Intel MEBX Password: Intel@1234 (password is case sensitive).
  • Leave the power states and user consent settings as is.

In the USB Drive section, browse to the USB drive.

Click OK.

The following warning appears.

Click Yes if you are sure you want to continue.

Ignore the error message shown while configuring the USB

The following dialog box appears.

Click OK.

You are now ready to configure the Intel vPro system.

Now that you have create a USB drive that can install the Intel AMT agent. You will need to reboot your computer with the USB drive inserted. When the system reboot you will be asked if you wish to install the Intel AMT agent

  1. Re-start your system.
  2. The system will read the USB drive if it is enabled to do so in the BIOS. If the system does not read
    3. the USB drive, check the BIOS.
  3. While the system is reading the USB drive, select Y when prompted to provision the system.
    4. Note: Provisioning might take about 15 seconds.

To configure a browser to connect, follow these steps:

  1. Go to command prompt and type ipconfig.
  2. Note down the ip address of your NUC machine.
  3. Go to next step to connect to your NUC from your laptop.

Now you are ready to start a web browser.

Point the web browser to the address. You may not able to access your own machine. Hence, try to connect to your neibour and see the result. http://192.168.__.__:16992

When prompted for username/password, type admin for username and Intel@1234 as the password

You are now connected. Please explore different sections.
