This guide is intended to be a starting point for testing your Intel ® vPro™ technology systems within your environment. It will guide you through setting up a "Sandbox" test environment using your own network environment so that you can test your specific use cases on Intel vPro technology that ultimately will help reduce overall support costs and costly service/truck rolls.

You need:

  • A host system (does not need to have Intel vPro technology)
  • An empty USB drive
  • An Intel vPro technology system with Intel® Active Management Technology (Intel® AMT) 7.0 or higher

To verify that the Intel vPro system has Intel AMT 7.0 or higher, look for one of these badges on the system case:

On the Intel vPro system:

  1. Open the Intel® Management and Security Status (IMSS) program.
  2. Click the Advanced tab.
  3. Under Intel® Management Engine, verify that the status is Unconfigured.

On the host system:

  1. Insert the empty USB drive in the USB port on the host system.
  2. Download Intel® Setup and Configuration Software (Intel® SCS) to the host system.
  3. Extract the Intel SCS download to a directory of your choice.
  4. Browse to the ACUWizard directory.
  5. Right-click ACUWizard.exe and select Run as administrator.

    6. The Intel® AMT Configuration Utility appears.

  6. Select Create Settings to Configure Multiple Systems.

    7. The following screen appears

  7. Click the Tools drop-down menu and select Prepare a USB Key for Manual Configuration

    8. The following screen appears

  8. Select the type of Intel vPro system that you will be configuring: Mobile Systems or Desktop Systems.

    9. Note: Some Small Form Factor (SFF) desktops use mobile chipsets. For example, QM87 is a mobile Intel vPro chipset noted by the “M,” whereas the desktop equivalent chipset is the Q87.

  9. Select All systems are version 7.0 and higher.
  10. In the Configuration Settings section:

    • Old Intel MEBX Password: Intel@1234

      Intel SCS always puts the default password of unconfigured systems ("admin") in this field.

      For the present workshop it is "Intel@1234"

      If this is not the password currently defined in Intel MEBX, enter the correct password.

      Note: If you have lost your Intel MEBX password, you will need to consult the OEM manual for clearing the CMOS battery to reset the password to "admin".

    • New Intel MEBX Password: type a new password. Click "i" for password rules.
    • Leave the power states and user consent settings as is.
  11. In the USB Drive section, browse to the USB drive.
  12. Click OK.

    13. The following warning appears.

  13. Click Yes if you are sure you want to continue.

    14. The following dialog box appears.

  14. Click OK.
  15. Remove the USB drive.

    16. You are now ready to configure the Intel vPro system.

  1. Move back to the Intel vPro system.
  2. Start with the system in the off state.
  3. Insert the newly setup USB drive in the USB port.
  4. Turn on the system.
  5. The system will read the USB drive if it is enabled to do so in the BIOS. If the system does not read
    6. the USB drive, check the BIOS.
  6. While the system is reading the USB drive, select Y when prompted to provision the system.

    7. Note: Provisioning might take about 15 seconds.

  7. To verify, do the following:

    8. a. Open the Intel® Management and Security Status (IMSS) program.

    b. Click the Advanced tab.

    c. Under Intel® Management Engine, verify that the status is Configured.

You have successfully configured the system

Now you can test your remote manageability using the host system and the Intel vPro system.

  1. Move back to the host system.
  2. On the host system, download and install the RealVNC* Viewer Plus trial version.
  3. Open the program.
  4. Enter the following information:
    • AMT Server: The of the Intel vPro system you are connecting to
    • Encryption: None
    • Connection Mode: Intel® AMT KVM
    • Click Options.
    • Match the items selected below and click OK.
    • On the VNC Viewer Plus screen, click Connect.
    • Log in to the Intel vPro system.
      • Username: admin
      • Password: <Your Intel MEBX password>
    • Move to the top of the window to use the menu.

You have successfully configured the system
