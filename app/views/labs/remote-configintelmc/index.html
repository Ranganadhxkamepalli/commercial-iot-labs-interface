Read the Overview

The web based Intel AMT management console is now called Mesh Commander. It runs on many operating systems and can be hosted in a web server.

With HTML5 being very capable, it's a lot easier for administrators to use web applications that interact with Intel AMT within a browser, making the local installation of tools a thing of the past in some cases.
