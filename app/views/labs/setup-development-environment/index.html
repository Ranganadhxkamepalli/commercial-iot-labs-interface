Lab Overview and Objectives

The labs in the Intel Visual Retail Workshop are organized in a progression that will result in each attendee building their own automated and secured, multi-device IoT network supporting Visual Retail scenarios.

Start by making sure your computer, is ready for Visual Retail development.

By the end of this module, you should be able to:

  • Unbox and setup your Intel® NUC Hardware
  • Connect to your Intel® NUC to power and monitor.
  • Install Ubuntu 16.04
  • Install Media Server Studio
  • Install OpenCV with Python support
  • Copy the required lab exercises

Hardware Requirements

5th Generation Core Processor


Software Requirements

Open network is recommended while running the CVSDK & OpenCV installations as corporte proxy blocks downloading certain files.

  • Install Ubuntu 16.04 from http://releases.ubuntu.com/16.04/
  • Note: While installing, select the option "install Ubuntu alongside with Windows"

    • Set name: intel

    computer name: intel<NUC Number>

    Username: intel

    Password: intel123


  • Update the Ubuntu packages:
    $sudo apt-get update

    Note: Troubleshoot for core dump error on update- $sudo apt-get purge libappstream3

  • Install Media Server Studio for Linux from https://software.intel.com/en-us/intel-media-server-studio

    Note: Use Ctrl+Alt+F1 to switch to one of the classic UNIX terminals and install the above SDK. It won't install properly in GUI based terminal. If you wan't to come back to GUI, press Ctrl+Alt+F7

    • Reboot the system

  • Install OpenCV with Python from https://opencv.org/releases.html

1. Run python

$idle
from terminal. Type: import cv2

2. It should not give any error.

3. Type: print cv2.__version__

4. It should display the cv2 version

5. Execute $vainfo and check if there are no errors

6. Execute

$cd /opt/intel/mediasdk/samples/_bin/x64

$./sample_decode h264 -i ../content/test_stream.264 -hw -vaapi -r

This should run a video and show the usage of hw and vaapi for decoding frames in terminal.

5. Shutdown the system, unplug the cables and stick Intel logo provided in the box, on to the NUC and pack the box.
