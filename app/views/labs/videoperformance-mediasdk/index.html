    • Note:
  • MS Visual studio 2015 is required for compiling this sample app.
  • Readme file specifies the supported input/output formats, HW/SW limitations etc.
  • Open the simple_sample Solution in MS Visual Studio 2015, build and run.
  • Command line for SW decode-
    • sample_decode.exe h264 -i C:\City.h264 –r -sw
  • Command line for HW decode-
    • sample_decode.exe h264 -i C:\City.h264 –r -hw
  • Open the task manager and compare CPU performance in each of the above cases.
  • Just type sample_decode.exe and explore other command line switches:
    -o : output to file in RAW formats
    -r: Present
    -f : change rendering frame rate
    -w and –h : changing the resolution
    etc
