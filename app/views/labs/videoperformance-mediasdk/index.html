  • Sample apps are already pre-compiled and executables are made available.
  • Go to C:\Users\<user name>\Documents\Intel Media SDK Samples 2016 6.0.0.142\_bin\x64
  • Type "cmd" in the navigation pane.

    • Note:
  • MS Visual studio is required for compiling the sample apps. Details in "Media Samples Guide".
  • Readme file specifies the supported input/output formats, HW/SW limitations etc.
  • Command line for SW decode-
    • sample_decode.exe h264 -i ..\content\sample.h264 –r -sw
  • Command line for HW decode-
    • sample_decode.exe h264 -i ..\content\sample.h264 –r -hw
  • Open the task manager and compare CPU performance in each of the above cases.
  • Just type sample_decode.exe and explore other command line switches:
    -o : output to file in RAW formats
    -r: Present
    -f : change rendering frame rate
    -w and –h : changing the resolution
    etc
  • Command line-
    • sample_video_wall.bat ..\content\sample.h264

    • Note:
  • No. of channels (i.e VerticalCells and HorizontalCells) can be edited in sample_video_wall.bat file.
  • Change MediaSDKImplementation to sw and hw.
  • Change these variable and notice the difference.
